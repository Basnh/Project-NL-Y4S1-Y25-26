Sửa file cấu hình ngay trên giao diện web có kiểm tra cú pháp.

Câu hỏi thường gặp

❓ Cần quyền gì để sử dụng UControl?

Tài khoản cần có quyền sudo trên máy chủ Ubuntu để thực hiện một số thao tác nhất định.

🔐 UControl có hỗ trợ SSH Key không?

Có, bạn có thể kết nối bằng SSH Key để bảo mật tốt hơn so với dùng mật khẩu.

🌍 Có thể quản lý bao nhiêu máy chủ?

Bạn có thể quản lý nhiều máy tùy nhu cầu. UControl hỗ trợ thao tác đa máy mượt mà.
